ตัวอย่าง Alignment Text

Text ในที่นี้หมายถึง อะไรก็ตามที่มีบนแป้นพิมพ์นั่นเอง ยกตัวอย่าง text ตัวอักษร สัญลักษณ์ เครื่องหมายต่างๆ เหล่านี้ล้วนเป็น text ในเนื้อหาส่วนนี้เราจะมาพูดถึงเรื่องของ Text และคุณสมบัติที่สามารถเพิ่มให้กับ Text ได้ ซึ่งหลักๆแล้วประกอบไปด้วย Color สีตัวหนังสือ Alignment ตำแหน่งของ text Decoration Transformation Spacing Shadow
